
# 🗄 DPP: SQL Practice Assignment – Basics to Constraints

## ⚠ Important Note

This folder contains SQL scripts and query files.
Please open the `.sql` file in MySQL / PostgreSQL to execute and test the queries.

---

## 📌 Overview

This practice assignment focuses on building foundational SQL skills, including:

* Database setup
* Table creation
* Data insertion
* Data retrieval
* Data modification
* Filtering and logical operators
* Data integrity constraints

The assignment simulates real-world employee database management scenarios.

---

# 🗂 Module Breakdown

---

## 🔹 1. Introduction to SQL & Basic Queries

### Objectives:

* Install MySQL or PostgreSQL
* Create a database (`company_db`)
* Create `employees` table
* Insert sample records
* Retrieve data using SELECT

### Skills Developed:

* Database creation
* Table schema design
* Basic SELECT queries

---

## 🔹 2. Filtering Data Using WHERE Clause

### Tasks Covered:

* Retrieve employees from specific department
* Filter salary using comparison operators
* Apply AND / OR conditions
* Use DISTINCT to retrieve unique departments

### Skills Developed:

* Data filtering
* Logical conditions
* Query refinement

---

## 🔹 3. Modifying Data (INSERT, UPDATE, DELETE)

### Tasks Covered:

* Insert multiple rows
* Update employee salary
* Delete specific employee
* Verify changes using SELECT

### Skills Developed:

* Data manipulation
* Transaction control
* Query validation

---

## 🔹 4. Using Constraints (NOT NULL, UNIQUE, DEFAULT, CHECK)

### Tasks Covered:

* Create `employees_v2` table with constraints
* Apply NOT NULL, UNIQUE, and CHECK constraints
* Test constraint violations (e.g., duplicate email entry)

### Skills Developed:

* Data integrity management
* Constraint-based validation
* Structured schema design
