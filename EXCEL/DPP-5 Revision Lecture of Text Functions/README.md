## 📌 Overview

This assignment focuses on applying Excel text functions for data cleaning, formatting, and transformation.

The tasks simulate real-world data preparation scenarios involving:

* Name standardization
* Email cleaning
* Product code formatting
* Domain extraction
* Identifier creation

The module strengthens business-ready text manipulation skills required in data analytics.

---

# 🗂 Module Breakdown

---

## 🔹 1. Name Cleaning & Standardization

### Tasks Covered:

* Remove extra spaces using TRIM
* Convert names to proper case using PROPER
* Extract first name
* Extract last name

### Skills Developed:

* String formatting
* Substring extraction
* Data normalization

---

## 🔹 2. Email Cleaning & Standardization

### Issues Addressed:

* Extra spaces before/after "@"
* Double spaces
* Uppercase characters
* Misplaced dots

### Tasks Covered:

* Create Clean_Email column
* Standardize emails using TRIM, LOWER, SUBSTITUTE
* Extract domain name (e.g., gmail.com, yahoo.com)

### Skills Developed:

* Data cleansing logic
* Nested text functions
* Domain extraction

---

## 🔹 3. Product Name & Code Standardization

### Tasks Covered:

* Convert product names to uppercase
* Standardize product codes into uniform format (e.g., P-001, P-005, P-008)
* Extract numeric portion of product code using TEXT function

### Skills Developed:

* Code formatting
* Numeric extraction from text
* Consistent identifier formatting

---

## 🔹 4. Text Search & Filtering

### Task:

* Identify entries containing the word "road" in address column

### Skills Developed:

* Keyword search logic
* Text filtering

---

## 🔹 5. Unique Identifier Creation

### Task:

Create a unique customer identifier in format:

> FirstName_Last3Digits
> Example: Alice_004

### Skills Developed:

* Text concatenation
* Substring extraction
* Automated ID generation
