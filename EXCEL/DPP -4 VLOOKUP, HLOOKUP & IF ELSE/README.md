## 📌 Overview

This assignment focuses on mastering Excel lookup functions and logical decision-making formulas.

The module includes:

* Vertical lookup (VLOOKUP)
* Horizontal lookup (HLOOKUP)
* Conditional logic using IF
* Nested logic using IF + AND / OR
* Dynamic designation generation
* Zone-based employee classification

This assignment strengthens structured data retrieval and business rule implementation in Excel.

---

## 🔹 Section 1: VLOOKUP Applications

### Tasks Covered:

* Fetch salary using employee name
* Retrieve department using Emp_ID
* Find city based on employee name
* Fetch score using Emp_ID
* Identify bonus eligibility status

---

## 🔹 Section 2: HLOOKUP Applications

### Tasks Covered:

* Retrieve salary after converting dataset horizontally
* Fetch department using Emp_ID

---

## 🔹 Section 3: IF Logic Implementation

### Salary Classification Rule:

* Salary ≥ 70,000 → "High Salary"
* Otherwise → "Regular Salary"

---

## 🔹 Section 4: IF + AND Logic

### Performance Rating Rule:

* Score ≥ 85
* Salary ≥ 60,000
* Otherwise → "Average Performer"

---

## 🔹 Section 5: Dynamic Designation Creation

Generate dynamic output in format:

* Mumbai Marketing Staff
* Delhi Finance Staff
* Bengaluru HR Staff

Without using multiple IF statements.

---

## 🔹 Section 6: IF + OR Logic (Zone-Based Designation)

City-to-Zone mapping:

* Mumbai, Pune → West Zone
* Delhi → North Zone
* Chennai, Bengaluru → South Zone

Output format:

> "<Zone> – <Department>"
